# Android-Clean-Arch-Coroutines-Koin

A Simple Android Mobile Application which has been implemented using Clean Architecture alongside MVVM design.


### Technologies & Methodologies which used:

- Koin
- Coroutines
- Clean Architecture
- MVVM Pattern
- LiveData
- Espresso
- Mockk
